Bookxlate Privacy Policy

Last Updated: June 22, 2026

Overview

Bookxlate is a Chrome extension that allows users to perform OCR on pages from Internet Archive books and translate the extracted text using Google's Gemini API.

Information Processed

When you choose to OCR or translate a page, the extension may access:

  • The currently viewed Internet Archive book page.
  • Images of book pages required for OCR processing.
  • Text extracted from those images.
  • Translation requests generated from the extracted text.

The extension processes only the content necessary to provide OCR and translation functionality.

Third-Party Services

To perform OCR and translation, the extension sends page images and/or extracted text to Google's Gemini API.

Information submitted to Gemini is subject to Google's terms and privacy practices. Users should review Google's privacy documentation for additional information.

Data Storage

The extension may store user preferences, settings, and API configuration information using Chrome's storage APIs.

This information is stored locally in the user's browser and is used solely to support the extension's functionality.

This extension requires a Google Gemini API key to function. The user must create their own key and input it to the extension. The key is stored locally in your browser via chrome.storage.local and is never sent anywhere except directly to Google's API. It is used solely to authenticate requests to Google's Gemini API.

Data Sharing

The extension does not sell, rent, or share personal information with third parties except as necessary to provide OCR and translation services through Google's Gemini API.

Data Retention

The extension does not maintain its own servers and does not intentionally retain submitted book content after processing. Any retention of submitted content is governed by Google's handling of requests to the Gemini API.

Permissions

activeTab

Used to access the current tab when the user explicitly invokes the extension.

scripting

Used to execute scripts required for OCR and translation functionality within supported pages.

storage

Used to save user settings and preferences.

Host Permissions

The extension requires access to Internet Archive pages in order to retrieve book page content and access to Google's Gemini API in order to perform OCR and translation.

Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be posted on this page.
