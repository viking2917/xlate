Bookxlate Privacy Policy

Last Updated: June 22, 2026

Overview

Bookxlate is a Chrome extension that allows users to perform OCR on pages from Internet Archive books and translate the extracted text using Google's Gemini API. It is a product of The Hawaii Project, LLC.

Information Processed

When you choose to OCR or translate a page, the extension may access:

  • The currently viewed Internet Archive book page.
  • Images of book pages required for OCR processing.
  • Text extracted from those images.
  • Translation requests generated from the extracted text.

The extension processes only the content necessary to provide OCR and translation functionality.

User Data collection, handling, storage and sharing

Data Storage

This extension requires the user to provide a Gemini API key. This API key used solely to authenticate requests to Google's Gemini API. It is stored in local storage in the browser, and only transmitted to the Google Gemini API, and only for the purpose of requesting OCR/translation of the viewed page.

No other personally identifiable user data is collected or stored.

The extension may store user app preferences, settings, and API configuration information using Chrome's storage APIs. These are not personally identifiable data. They are not transmitted to any other location or service.

Data Sharing

The extension does not sell, rent, or share personal information with third parties except as necessary to provide OCR and translation services through Google's Gemini API.

Data Retention

The extension does not maintain its own servers and does not retain submitted book content after processing. Any retention of content submitted to Google's API is governed by Google's handling of requests to the Gemini API.

Permissions

This extension requires the following permissions:

activeTab

Used to access the current tab when the user explicitly invokes the extension.

storage

Used to save user settings and preferences for the app in local storage.

Host Permissions

The extension requires access to Internet Archive pages in order to retrieve book page content and access to Google's Gemini API in order to perform OCR and translation.

Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be posted on this page.

Third-Party Services

To perform OCR and translation, the extension sends page images and/or extracted text to Google's Gemini API.

Information submitted to Gemini is subject to Google's terms and privacy practices. Users should review Google's privacy documentation for additional information.

Disclaimer

This extension is in no way associated with, endorsed by, or a product of, the Internet Archive organization or Google.
